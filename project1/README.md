# 📊 Excel Salary Dashboard

An interactive Excel dashboard that analyzes salaries for data-related jobs using the 2023 Data Jobs Salary dataset. The dashboard allows users to explore salary trends based on job title, country, and employment type, helping users understand compensation patterns across the global data industry.

---

## 📌 Project Overview

This project demonstrates how Microsoft Excel can be used to transform raw data into an interactive dashboard for data analysis.

Users can:
- Compare median salaries across different job roles
- Analyze salaries by country
- Filter results using interactive dropdowns
- Explore global salary trends through visualizations

---

## 📂 Dataset

The dashboard uses the **2023 Data Jobs Salary Dataset**, which includes information such as:

- Job Title
- Salary
- Country
- Employment Type
- Company Location
- Experience Level

---

## 📈 Dashboard Features

### 💼 Salary by Job Title

Displays the median salary for different data-related job roles using a horizontal bar chart.

### 🌍 Salary by Country

Shows salary distribution across different countries using an Excel Map Chart.

### 🎛 Interactive Filters

Users can filter the dashboard by:

- Job Title
- Country
- Employment Type

making it easy to explore customized salary insights.
